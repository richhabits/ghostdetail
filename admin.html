📸 R2 PHOTO STORAGE — Cloudflare R2 bucket: ghost-detail-media ✅ Created
To enable photo uploads: Go to Cloudflare Dashboard → R2 → ghost-detail-media → Manage R2 API tokens → Create token with "Object Read & Write" → Add R2_ACCESS_KEY_ID and R2_SECRET_ACCESS_KEY below. Also enable Public Access on the bucket and CNAME media.ghostdetail.autos to the public URL.
📅 GOOGLE CALENDAR — Auto-sync bookings
Get a token: OAuth Playground → Select Calendar API v3 → Authorize → Exchange for token → Add as GOOGLE_CALENDAR_TOKEN. Bookings auto-create in your calendar on confirmation.
💬 WHATSAPP BUSINESS API — ghost-whatsapp function ✅ Deployed
Add WHATSAPP_TOKEN (Meta Business → WhatsApp → API Token) and WHATSAPP_PHONE_ID (your business phone number ID). Enables: booking confirmations, deposit chases, review requests, MOT reminders, ceramic anniversaries. All fire automatically from followup_queue.
