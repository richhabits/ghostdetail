# Ghost Detail Autos
**ghostdetail.autos** — Luxury automotive detailing studio, Watford WD18

## Stack
- **Frontend**: Static HTML/CSS/JS — deployed on Vercel
- **Backend**: Supabase (33 edge functions, 112 tables, London region)
- **Payments**: Stripe
- **Email**: Resend
- **AI**: Ghost AI Router (multi-tier)
- **Media**: Cloudflare R2

## Key Pages
- `/` — Homepage with DVLA reg lookup
- `/studio` — Services & pricing
- `/concierge` — MOT management, PDR, alloys  
- `/paint-check` — Ghost Vision AI paint assessment
- `/booking` — 4-step booking form
- `/garage` — Customer vehicle portal
- `/admin` — Owner admin panel (PIN: see env)
- `/ghost-war-room` — AI war room
